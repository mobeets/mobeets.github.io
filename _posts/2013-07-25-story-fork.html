--- layout: post title: "Story fork" description: "" category: coded tags: [] --- {% include JB/setup %} Story fork allows users to build non-linear conversations with tweets.

Story fork was conceived as a platform for writing collaborative branching stories via twitter, but it can also be used as a scaffolding for twitter-based discussions, given that twitter's support for "conversations" is extremely limited.

From a more technical perspective, story fork displays tweets as trees using the javascript library d3.
